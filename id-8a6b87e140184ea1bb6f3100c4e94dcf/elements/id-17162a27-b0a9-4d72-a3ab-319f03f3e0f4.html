Conduzir Comunicação com Clientes ()
A atendente pode realizar chamadas telefonicas para conduzir comunicações diretas e pessoais com os clientes com o objetivo de resolver as pendencias de pagamento de forma rápida, discussões detalhadas, negociações de pagamento ou esclarecimento de dúvidas. O Atendente também pode usar e-mails para enviar informações detalhadas, extratos de conta, notificações de atraso ou acordos de pagamento formalizados. Tais registros escritos podem ser úteis para ambas as partes em futuras referências. SMS também pode ser usados para enviar lembretes rápidos ou alertas sobre pagamentos pendentes. Esta forma de comunicação é útil para alcançar clientes que podem não usar e-mail ou aplicativos de mensagens regularmente. No caso de avisos legais ou comunicações formais importantes, a correspondência física é utilizada. Este método pode ser necessário para cumprir requisitos legais ou para clientes que preferem comunicações tradicionais.
